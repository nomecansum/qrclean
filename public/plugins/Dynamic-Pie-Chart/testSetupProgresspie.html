Test and demonstration of the replace parameter of setupProgressPie

Original Setting:

$(".pie").setupProgressPie({
    color: "navy",
    size: 50,
    strokeWidth: 0,
    ringWidth: 5,
    rotation: true,
    valueAdapter: function(percent) { return 80; }
});

60

Merge options

No boolean argument (replace === undefined)

$(".pie.mergeOptions1").setupProgressPie({
    color: "red",
    strokeWidth: 1
});

60

Argument replace === false

$(".pie.mergeOptions2").setupProgressPie({
    color: "red",
    strokeWidth: 1
}, false);

60

Replace options

Argument replace === true

$(".pie.replaceOptions").setupProgressPie({
    color: "red",
    strokeWidth: 1 
}, true);
